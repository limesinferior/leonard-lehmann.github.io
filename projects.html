Casino Web-App

In A Group of 4 people we developed an online-casino featuring Black-Jack and Roulette. Further features included A user-management system with different User-Roles like the owner of the casino and clients. Individual user-game statistics where sampled and could be tracked with an integrated visualization tools. The Whole application was implented with Spring MVC, whereas the page structure was developed in HTML and CSS. We documented the whole process in a developers manual including UML diagrams such as class-diagrams, sequence diagrams, Use-Case diagrams and so on.

Tech Stack:

Spring MVC

Java

HTML

CSS3

Android-Application for Visually Handicapped People

Together with a friend i developed an android-app for visually handicapped people. The aim was to support visually handicapped people in making horizontal photos via speech-instructions.

When making photos the user was assisted in balancing the mobile phone horizontally right above the document via speech instructions.The snapshot was then transformed to a pdf-file.

Therapy Decision-Support-System

In my studies it is necessary as a preparation for the diploma-thesis to write a small thesis. Since I am interested in applications of machine-learning I decided to challenge myself with a topic that was offered by the IBMT-institute at the TU Dresden : "Development and Evaluation of a probabilistic Recommender-System for Therapy-Decision-Support".

The aim was to implement and evaluate a bayes-net based recommender-system for a database of psoriasis-patients. Everything was implemented in Python. Unfortunately I am not able to share the whole project here due to privacy-concerns in the patient-database. But in a blogpost i will further explain what I exactly implemented. Furthermore there was no Python implementation of a Naive-Bayes algorithm featuring different attribute scales together in any python library, so I implemented one on my own.

Tech Stack:

Python
